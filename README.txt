# Installation_Manual
1) Download UniServer:
Visit the UniServer download page: UniServer Download
Choose the version you want to download (e.g., UniServer Zero XIV).
2) Run the Installer:
After the download is complete, run the installer. Follow the on-screen instructions.
3) Choose Installation Directory:
Select the directory where you want to install UniServer. The default directory is usually fine.
4) Select Components:
Choose the components you want to install. The default components are Apache, MySQL, and PHP. You can customize this based on your needs.
5) Configure MySQL:
During the installation, you will be prompted to set a MySQL root password. Enter a secure password and remember it, as you will need it later.
6) Complete Installation:
Complete the installation process. UniServer will copy files to the specified directory and configure the services.
7) Start UniServer:
After installation, navigate to the installation directory (e.g., C:\UniServerZ) and find the executable file named UniController.exe.
Run UniController.exe to start the control panel.
8) Start Apache and MySQL:
In the UniController control panel, you'll see options to start Apache and MySQL. Click on the "Start" buttons next to both Apache and MySQL.
9) Access Web Server:
Open your web browser and go to http://localhost. If everything is set up correctly, you should see the default UniServer page.
10) Access phpMyAdmin:
To access phpMyAdmin (MySQL administration tool), go to http://localhost/phpmyadmin. Log in with the MySQL root username and the password you set during installation.
